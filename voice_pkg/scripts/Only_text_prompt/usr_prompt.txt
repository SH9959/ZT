你只可以用以下回答来输出以指挥机器人做动作，这些回答包括以下9个动作,前面数字是对应的序号：

[
1:招手
2:介绍自己
3:介绍右方
4:介绍右下方
5:介绍较近的前方
6:介绍较远的前方
7:回忆 刚刚我们
8:复杂的讲解动作
9:结束姿势
]

请根据文本的情境，匹配出对应的动作来回答，以下是一些可能示例：
[
    {
        输入：‘你好’或者‘再见’
        你需要输出：1:招手
    }
    {
        输入：你是谁呀，可以介绍一下吗
        你需要输出：2:介绍自己
    }
    {
        输入：可以介绍一下右下方的物品是什么吗
        你需要输出：3:介绍右下方
    }
    {
        输入：在我们右边的展品是...
        你需要输出：4:介绍右右方
    }
    {
        输入：位于我们面前的展品是...
        你需要输出：5:介绍较近的前方
    }
    {
        输入：介绍一下前面远处的展品
        你需要输出：6:介绍较远的前方
    }
    {
        输入：刚才我们讲的是...
        你需要输出：7:回忆 刚刚我们
    }
    {
        输入：‘正在讲解一段很长的文本’
        你需要输出：8:复杂的讲解动作
    }
    {
        输入：你摆一个POSE,可以合影吗
        你需要输出：9:结束姿势
    }
]

如果你认为输入文本没有上述的动作意图，你需要输出‘10:没有合适的动作’，以下是2个可能的例子：
[
    {
        输入：今天的天气怎么样
        你需要输出：10:没有合适的动作

        输入：你吃饭了吗
        你需要输出：10:没有合适的动作
